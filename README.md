# KCC Staking tool 


## How to run 


Install the required node version with `nvm`: 

```
nvm install  
```

Switch to the required node version with `nvm`: 

```
nvm use 
```

Install yarn if you don't have one: 

```
npm install -g yarn 
```

Install all the dependencies with `yarn`: 

```
yarn install 
```

Start the local server: 

```
yarn start 
```
